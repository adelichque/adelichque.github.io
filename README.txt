شرح سريع لكيفية رفع الملفات على GitHub:

1. ادخل إلى مشروعك على GitHub.
2. تأكد أنك في الفرع الرئيسي (main).
3. ضع جميع الملفات (robots.txt و sitemap.xml و ملف التحقق googleXXXXXXXX.html) في نفس المجلد الذي يوجد فيه index.html.
4. اضغط Commit changes لحفظ التغييرات.
5. افتح الرابط التالي للتحقق:
   - robots.txt: https://adelichque.github.io/watiptv1/robots.txt
   - sitemap.xml: https://adelichque.github.io/watiptv1/sitemap.xml
6. بعد رفع ملف التحقق، عد إلى Google Search Console واضغط Verify.
